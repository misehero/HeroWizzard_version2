Role overview
Admin
Full access — CRUD, backup/restore, lookups, user management, audit logs, Excel exports.
Manager
Everything an Accountant can do, plus rule creation, management, and Excel exports.
Accountant
CSV import + full transaction access + Excel export. Cannot manage rules.
Viewer
Read-only. Can view transactions and export Excel. No imports, rules, or lookups.

Frontend pages
Dashboard (dashboard.html)
Transaction table with filtering, sorting, pagination. Bulk update. Add/Edit modal. Audit log modal. Server backup restore. Export Excel.
Import CSV (upload.html)
CSV upload with drag & drop. Auto-detect bank format (Generic, Raiffeisen, Creditas). Import history.
Category Rules (categories.html)
6-level match hierarchy. Match modes: contains, exact, starts_with. Auto-set fields including projekt/produkt/podskupina. Export Excel.
Lookups (lookups.html)
Admin-only. 3 tabs: Projekty, Produkty, Podskupiny. Inline editing, reordering, soft delete. Export Excel (3 sheets).
Users (users.html)
Admin-only. User CRUD, role management, password set/reset. Last-admin protection. Export Excel.
Login (index.html)
Email + password. Forgot password link (shown after failed login). Pre-fills email.
