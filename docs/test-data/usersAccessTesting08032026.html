Role overview
Admin
Full access — CRUD, backup/restore, lookups, user management, audit logs.
Manager
Everything an Accountant can do, plus rule creation and management.
Accountant
CSV import + full transaction access. Cannot manage rules.
Viewer
Read-only. Can view/edit transactions and export data. No imports or rules.
